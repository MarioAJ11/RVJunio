# RV-T4P1 — Practica 4.1

Copia portable del proyecto Godot. Abrir con **Godot 4.6** en modo **Compatibility**.

## Contenido incluido

- `scenes/main.tscn` — escena principal
- `scripts/` — logica malabares, XR, simulador PC
- `objects/` — pelotas pickable
- `addons/` — godot-xr-tools + godotopenxrvendors (con binarios)
- `project.godot`, `export_presets.cfg`, `openxr_action_map.tres`

## Primer arranque

1. Godot 4.6 → Import → esta carpeta
2. Modo Compatibility
3. Abrir `scenes/main.tscn` → F6

Para export APK: instalar Android Build Template en el PC destino (crea `android/`).

Instrucciones completas: `../README.md`

# RV T2P2 - Sala interacciones VR (Unity 6.3)

Practica 2.2 del Tema 2: proyecto VR completo con Meta Quest.

## Que incluye

- Escena `Assets/VRInteractions.unity` (Camera Rig, grab, suelo, cubos)
- Meta XR SDK 203 + OpenXR Android/PC
- `ScriptUI.cs` + `VRUiBootstrap.cs` (UI runtime, boton B)
- URP Mobile, AndroidManifest Quest, build settings

## Abrir

1. Unity Hub → **6000.3.18f1** → Add project
2. Meta → Meta XR Simulator → Activate
3. Abrir `VRInteractions.unity` → Play

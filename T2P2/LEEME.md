# RV-T2P2 — Practica 2.2

Copia portable del proyecto Unity. Abrir con **Unity Hub 6000.3.18f1**.

## Contenido incluido

- `Assets/` — escenas, scripts, XR
- `Packages/` — manifest y dependencias Meta XR
- `ProjectSettings/` — configuracion del proyecto

## Primer arranque

1. Unity Hub → Add → esta carpeta
2. Esperar importacion (crea `Library/` automaticamente)
3. Meta XR Simulator para probar en PC

Instrucciones completas: `../README.md`
